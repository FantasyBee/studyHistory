### 第一种：对String类型的集合

```
ArrayList<String> list = new ArrayList<>();
```

```
import java.util.ArrayList;
import java.util.Collections;

public class Main {
    public static void main(String[] args) {
        // 创建一个 ArrayList 对象
        ArrayList<String> list = new ArrayList<>();

        // 添加元素到 ArrayList，list.add("Apple");
        list.add("Banana");
        list.add("Orange");
    
        // 遍历 ArrayList
        System.out.println("ArrayList 中的元素:");
        for (String fruit : list) {
            System.out.println(fruit);
        }
    
        // 获取 ArrayList 的大小
        System.out.println("ArrayList 的大小: " + list.size());
    
        // 检查 ArrayList 是否为空
        System.out.println("ArrayList 是否为空: " + list.isEmpty());
    
        // 获取 ArrayList 中的元素
        System.out.println("第二个元素是: " + list.get(1));
    
        // 修改 ArrayList 中的元素
        list.set(1, "Grapes");
        System.out.println("修改后的第二个元素是: " + list.get(1));
    
        // 删除 ArrayList 中的元素
        list.remove(0);
        System.out.println("删除第一个元素后:");
        for (String fruit : list) {
            System.out.println(fruit);
        }
    
        // 排序 ArrayList
        Collections.sort(list);
        System.out.println("排序后:");
        for (String fruit : list) {
            System.out.println(fruit);
        }
    
        // 清空 ArrayList
        list.clear();
        System.out.println("清空后, ArrayList 是否为空: " + list.isEmpty());
    }

}
```

### 第二种：对数组类型的集合

```
ArrayList<Integer> numbers = new ArrayList<>();
```

```
import java.util.ArrayList;
import java.util.Collections;

public class SortArrayList {
    public static void main(String[] args) {
        ArrayList<Integer> numbers = new ArrayList<>();
        numbers.add(5);
        numbers.add(3);
        numbers.add(8);
        numbers.add(1);
        
        System.out.println("排序前:");
        for (int num : numbers) {
            System.out.println(num);
        }
        
        Collections.sort(numbers);
        
        System.out.println("排序后:");
        for (int num : numbers) {
            System.out.println(num);
        }
    }
}

```

### 第五种：进制转换

```
在Java中，你可以使用内置的方法来进行进制转换。其中最常用的方法是使用包装类 Integer、Long、BigInteger 等的静态方法。以下是一些常见的进制转换方法：

十进制转其他进制：
十进制转二进制：Integer.toBinaryString(int num)
十进制转八进制：Integer.toOctalString(int num)
十进制转十六进制：Integer.toHexString(int num)

//实例
int decimal = 42;
String binary = Integer.toBinaryString(decimal);
String octal = Integer.toOctalString(decimal);
String hex = Integer.toHexString(decimal);

System.out.println("Binary: " + binary); // 输出: 101010
System.out.println("Octal: " + octal);   // 输出: 52
System.out.println("Hexadecimal: " + hex); // 输出: 2a


其他进制转十进制：
二进制转十进制：Integer.parseInt(String binaryString, int radix)
八进制转十进制：Integer.parseInt(String octalString, int radix)
十六进制转十进制：Integer.parseInt(String hexString, int radix)


String binary = "101010";
String octal = "52";
String hex = "2a";

int decimalFromBinary = Integer.parseInt(binary, 2);
int decimalFromOctal = Integer.parseInt(octal, 8);
int decimalFromHex = Integer.parseInt(hex, 16);

System.out.println("Decimal from binary: " + decimalFromBinary); // 输出: 42
System.out.println("Decimal from octal: " + decimalFromOctal);   // 输出: 42
System.out.println("Decimal from hexadecimal: " + decimalFromHex); // 输出: 42
```

### 第六种：数组与字符串与字符数组之间的转换

```
数组与字符串之间的转换：
将数组转换为字符串：使用 Arrays.toString() 方法。
将字符串转换为数组：使用 String.split() 方法或者逐字符遍历字符串。

// 数组转字符串
int[] array = {1, 2, 3, 4, 5};
String arrayString = Arrays.toString(array);
System.out.println("Array as string: " + arrayString); // 输出: [1, 2, 3, 4, 5]

// 字符串转数组
String str = "1,2,3,4,5";
String[] strArray = str.split(",");
System.out.println("String as array: " + Arrays.toString(strArray)); // 输出: [1, 2, 3, 4, 5]



字符串与字符之间的转换：
将字符串转换为字符数组：使用 toCharArray() 方法。
将字符数组转换为字符串：使用 String.valueOf(charArray) 或者 new String(charArray) 方法。

// 字符串转字符数组
String str = "hello";
char[] charArray = str.toCharArray();
System.out.println("String to char array: " + Arrays.toString(charArray)); // 输出: [h, e, l, l, o]

// 字符数组转字符串
char[] charArray = {'h', 'e', 'l', 'l', 'o'};
String str = String.valueOf(charArray);
System.out.println("Char array to string: " + str); // 输出: hello



字符串与数字之间的转换：
将字符串转换为数字：使用 Integer.parseInt(String str) 或者 Double.parseDouble(String str) 等方法。
将数字转换为字符串：使用 String.valueOf(int num) 或者 Integer.toString(int num) 方法。

// 字符串转数字
String str = "123";
int num = Integer.parseInt(str);
System.out.println("String to int: " + num); // 输出: 123

// 数字转字符串
int num = 123;
String str = String.valueOf(num);
System.out.println("Int to string: " + str); // 输出: 123
```
